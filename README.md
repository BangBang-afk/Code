# Shubang Srivatsa - Portfolio

A modern, interactive portfolio showcasing AI systems architecture, full stack development, and cloud infrastructure expertise.

## 🚀 Tech Stack

- **Framework:** Next.js 16 (React 19)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript
- **Deployment:** Vercel

## ✨ Features

- 🎨 Modern, minimalist dark theme design
- 📱 Fully responsive and mobile-optimized
- ✨ Smooth animations and transitions
- 🎯 Interactive contribution grid tracker
- 📊 Timeline of professional journey
- 🔗 Social media integration (GitHub, LinkedIn)
- 📧 Email contact integration
- ♿ Accessible component design

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/BangBang-afk/Code.git
cd Code

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

## 🛠️ Development

```bash
# Development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Run linter
npm run lint
```

## 🚢 Deployment

This project is optimized for deployment on **Vercel**.

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Vercel will auto-detect Next.js and deploy automatically

No additional configuration needed!
